# Hormone Duel — Solo Bot

เกม Static Web App เล่นคนเดียวกับบอท ไม่ใช้ Vercel, Supabase, Node.js หรือเซิร์ฟเวอร์เกม

## เปิดเล่น
เปิด `index.html` ได้โดยตรงบนคอมพิวเตอร์

## เล่นบน Android / ส่งให้คนอื่น
อัปโหลดโฟลเดอร์นี้ไปยัง Static Hosting เช่น GitHub Pages, Cloudflare Pages หรือ Netlify แล้วส่งลิงก์ให้ผู้เล่นได้เลย เมื่ออัปโหลดแล้วเจ้าของคอมปิดเครื่องได้ เพราะเกมทำงานใน Browser ของผู้เล่นเอง

## ระบบ
- ผู้เล่น vs บอท
- HP เริ่ม 100
- น้ำเริ่ม 100
- การ์ดเริ่ม 5 ใบ
- ทุกครั้งที่เริ่มเทิร์นจะจั่ว 1 ใบ
- ใช้การ์ดได้ 1 ใบต่อเทิร์น
- น้ำ < 20 เสีย HP 5 ต่อเทิร์น
- น้ำ < 0 เสีย HP 20 ต่อเทิร์น
- เด็คสร้างจากการ์ดฮอร์โมน 40 ชนิด และแต่ละชนิดมีได้ไม่เกิน 3 ใบ
