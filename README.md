# Hormone Duel v4

เกมเว็บแบบ Static สำหรับเล่นคนเดียวกับบอท

เพิ่มในเวอร์ชันนี้:
- เลือกการ์ดทั้งผู้เล่นและบอทก่อน แล้วจึงประมวลผลทั้งสองฝ่าย
- สุ่มผู้เริ่มประมวลผลในรอบแรก และสลับผู้เริ่มในรอบถัดไป
- มีอนิเมชันประกาศผู้เริ่มก่อนประมวลผล
- มีคู่มือครั้งแรกด้วย localStorage
- เด็คสูงสุด 60 ใบ
- เพิ่ม Melatonin, Oxytocin, hCG, Thymosin, Gastrin, Secretin, CCK และ EPO
- เพิ่มดีบัพหลับ
